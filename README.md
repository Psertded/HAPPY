## 中文

# HAPPY

是一个搞笑的程序。<br>
它可以:
1. 播放搞笑的的音乐 
2. 移动窗口本身
3. 移动你的鼠标
4. 在屏幕上显示"午夜Happy"

注：此程序不是 **病毒**！！！按下ESC退出。
